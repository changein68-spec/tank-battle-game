# 🎮 坦克大战 (Battle City)

经典 FC 红白机《坦克大战》的网页版复刻，纯 HTML/CSS/JavaScript 实现。

## 🎯 游戏特色

- **经典玩法** — 操控坦克保卫基地，消灭所有敌方坦克
- **3张精心设计的地图** — 还原经典布局，含砖墙、钢墙、水域、树林等地形
- **4种敌人类型** — 不同颜色和速度的敌方坦克
- **道具系统** — ⭐火力增强 / 🛡️护盾 / 💣全屏炸弹 / ⏰时间冻结 / 🏗️修复围墙
- **音效反馈** — Web Audio API 生成射击、爆炸等音效
- **双端支持** — 键盘操控 + 移动端触控

## 🕹️ 操作方式

| 操作 | 键盘 | 移动端 |
|------|------|--------|
| 移动 | `WASD` 或 `↑↓←→` | 左半屏滑动 |
| 开火 | `空格` 或 `J` | 右半屏点击 |
| 暂停 | `P` | — |
| 开始/重来 | `Enter` | 点击画面 |

## 🚀 快速开始

直接在浏览器中打开 `index.html` 即可游玩，无需任何依赖或构建工具。

```bash
# 克隆仓库
git clone https://github.com/YOUR_USERNAME/tank-battle.git

# 用浏览器打开
open index.html
# 或者直接双击 index.html
```

## 🛠️ 技术实现

- **Canvas 2D** — 游戏画面渲染
- **Web Audio API** — 实时音效合成
- **纯原生 JS** — 无框架、无依赖

## 📝 游戏规则

1. 操控坦克消灭所有敌方坦克即可过关
2. 保护地图底部的基地（鹰标志），基地被摧毁则游戏结束
3. 拾取道具提升战斗力
4. 每关敌人数量和难度递增
